# ng-notadd

基于 Angular7  Material2  的中后台解决方案

### 技术栈

- Typescript
- Angular
- Material2
- rxjs
- Graphql

#### 相关链接

[DEMO](https://notadd.github.io/ng-notadd)

[ng-notadd-mock-server](https://github.com/notadd/ng-notadd-mock-server)

## Quick start

```bash
    git clone https://github.com/notadd/ng-notadd.git
    
    cd ng-notadd
    
    npm install
    npm start
    # or use ng cli
    ng serve
```
